Steps Undertaken to Build App:


1 - Built simple web app: hello cloud app with music feature + email form submission feature


2 - Integrated static web hosting: used AWS Amplify to host static web resources like HTML, CSS, and image files


3 - Built serverless backend: created lambda function to send email upon form submission using Amazon SES


4 - Deployed a RESTful API: used Amazon API Gateway to make use of the Lambda function as a RESTful API


AWS Cloud Services Utilized:

AWS Amplify, Lambda, Amazon Simple Email Service, Amazon API Gateway, IAM
